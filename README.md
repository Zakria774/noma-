# 🚀 Noma – AI-Powered Financial Dashboard

Noma is a modern, AI-driven invoicing and expense management web application designed for freelancers, startups, and small businesses. It simplifies financial tracking by combining a clean dashboard, intuitive workflows, and a built-in AI assistant that understands natural language commands.

---

## ✨ Overview

Managing finances shouldn’t be complicated. Traditional accounting tools are often overwhelming, cluttered, and time-consuming.

**Noma solves this by:**

* Providing a clean, easy-to-read financial dashboard
* Allowing users to create invoices and track expenses effortlessly
* Using an AI assistant to perform actions through simple commands
* Giving real-time insights into business performance

---

## 🧠 Key Features

### 📊 Dashboard

* Revenue, expenses, and net result overview
* Visual graph of money in vs money out
* Smart AI insights (unpaid invoices, spending patterns, performance)
* Clear positive/negative financial status indicator

---

### 💰 Invoice Management (Sales)

* Create, edit, and delete invoices
* Invoice preview before sending
* Mark invoices as **paid / sent / draft**
* Highlight overdue invoices
* Search and manage all invoices in one place

---

### 🧾 Expense Management (Purchases)

* Add and manage business expenses
* Upload receipts
* Draft vs booked purchase logic
* Only booked purchases affect financial results
* Edit and categorize expenses easily

---

### 🔍 Purchase Review

* Preview uploaded receipts
* Review extracted data
* Send data into purchase drafts
* AI-assisted refinement

---

### ⚙️ Settings

* Business name and currency
* Default tax and invoice note
* Auto due date configuration
* Multiple theme modes (light, mint, soft, midnight)

---

### 🤖 AI Assistant (Core Feature)

Interact with the app using natural language:

Example commands:

* “Create invoice for Ali 5000 due next week”
* “Show unpaid invoices”
* “Add expense 200 for fuel”
* “Show profit”

**Capabilities:**

* Auto-create invoice drafts
* Navigate between sections
* Show financial insights
* Assist with expense tracking

---

## 🧮 Business Logic

* Revenue = Sent + Paid invoices
* Expenses = Only booked purchases
* Draft purchases do NOT affect totals
* Net Result = Revenue – Expenses
* Overdue invoices are highlighted and surfaced in insights
